How to Import Your Calendar

LewisCal lets you bring in events from other calendar apps so you can see your schedule in one place. You can import your calendar in two ways: using a calendar link or by uploading a .ics file.

Option 1: Import Using a Calendar Link

  1. In your calendar app (for example Google Calendar), find the option to get a link to your calendar in .ics format.
  2. Copy the calendar link to your clipboard.
  3. On the LewisCal home page, paste the link into the box labeled "Paste .ics calendar link here".
  4. Click the "Import Calendar" button.
  5. You will see a message like "Importing calendar..." while LewisCal loads your events. When it finishes, it will show how many events were loaded, and your events will appear under "Your Events".

If there is a problem with the link, you will see an error message instead, such as "Invalid calendar URL." or "Error loading calendar.".

Option 2: Import Using a .ics File

  1. In your calendar app, export your calendar as a .ics file and save it to your computer.
  2. On the LewisCal home page, find the section that says "Or upload an .ics file:".
  3. Click the file chooser (the "Choose file" button) and select your .ics file.
  4. Click the "Upload File" button.
  5. You will see a message like "Importing calendar file..." while LewisCal reads and processes the file. When it finishes, your events will appear under "Your Events".

If the file is not a valid .ics file or contains no events, LewisCal will show an error message so you know something went wrong.
